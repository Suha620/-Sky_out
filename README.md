# 苍穹外卖系统增强版

### 1. 缓存优化（Redis）
 使用 Redis 缓存首页分类数据、热门菜品数据
 设置合理过期时间 + 随机过期时间（防止缓存雪崩）
 使用逻辑过期解决缓存击穿问题
### 2. 分布式锁优化
基于 Redis 实现分布式锁（SETNX + 过期时间）
使用 Lua 脚本保证操作原子性
防止锁误删（使用唯一标识）
## 3. 异步处理（Kafka）
### 4. 定时任务优化（订单自动关闭）
使用 Kafka + 定时扫描机制
定时检查超过 3 天未支付订单并自动关闭
更新订单状态并释放资源
### 5. 数据库优化
添加合理索引（如订单状态、创建时间）
避免全表扫描
对复杂查询进行SQL优化
